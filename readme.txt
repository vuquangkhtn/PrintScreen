ScreenWatch. 
Write a program to periodically every 2 second save a picture of the screen to disk. Activate with Ctrl + Alt + Shift + S.
Filename be in this format: "Screen_yy-mm-dd hhmmss.bmp".


/////////////////////////////////////////////////////////
===Main flow===
Press Ctrl + Alt + Shift + S. Periodically every 2 second save a picture of the screen to "images" folder.


===Additional flow===

Click "Stop" to pause saving a picture of the screen.
Press Ctrl + Alt + Shift + S to start again.
